# Password Strength Checker

Project sederhana untuk mengecek kekuatan password berdasarkan panjang, kombinasi karakter, dan pola umum. Cocok untuk pembelajaran dasar keamanan siber dan validasi password pada aplikasi web.

## Fitur

- Input password secara langsung
- Analisis kekuatan password: Weak, Medium, atau Strong
- Pengecekan berdasarkan panjang, kombinasi karakter, dan pola umum
- Antarmuka sederhana


## Teknologi yang Digunakan

- HTML dan CSS  
- JavaScript (logika pengecekan)  

## Cara Menggunakan

1. Buka file `index.html` di browser.
2. Masukkan password pada input yang tersedia.
3. Hasil analisis kekuatan password akan ditampilkan secara otomatis.

## Catatan

- Project ini dibuat untuk tujuan pembelajaran.
- Tidak disarankan digunakan sebagai sistem validasi keamanan produksi tanpa perbaikan lebih lanjut.
